# チャート

> 文中の `[K1]` などは、その挙動を保証する E2E flow の ID。一覧は [`tests/e2e/FLOWS.md`](../../tests/e2e/FLOWS.md) を参照。

チャートウィンドウはローソク足（OHLC）を描画するフローティングウィンドウです。タイトルは `CHART — {instrument_id}` で、表示する銘柄ごとに 1 つ生成されます。チャートはサイドバーで選択中の銘柄から自動的に導出されます [K1]。共通のウィンドウ操作は [ウィンドウとパネル](windows-and-panels.md) を参照してください。

## 画面構成

| 領域 | 内容 |
| --- | --- |
| メイン領域 | ローソク足（OHLC）。`close >= open` の陽線は緑、`close < open` の陰線は赤。実体（ボディ）と高安を結ぶヒゲ（ウィック）を描画。 [K1] |
| 出来高サブペイン | メイン領域下部（描画領域の約 20%）にローソクと同色の半透明バーで出来高を表示。 [K1] |
| 右ガター | 価格軸（Y 軸）のラベル。 [K1] |
| 下ガター | 時間軸（X 軸）のラベル。 [K1] |
| 上部 | 現在価格の表示。 [F1]/[K1] |
| クロスヘア | カーソル位置に十字線を描画し、ガター内に価格・時刻のバッジを強調表示。出来高サブペイン上では出来高を表示。 [K1] |
| Ladder ペイン | Live モードのみ。詳細は下記を参照。 [K5] |

## 価格スケールとオートスケール

- 表示中のローソク足の高値・安値からメイン領域に収まるよう価格スケールを自動調整（オートスケール）します。 [K1]
- パン・ズーム操作を行うとオートスケールは自動的に無効化され、操作した表示が次フレームで巻き戻らないようになっています。 [K2]/[K3]
- ダブルクリックでオートスケールを再有効化できます（下記「操作」を参照）。 [K3]

## 操作

| 操作 | 効果 |
| --- | --- |
| ホイール | 縦方向のズーム（価格スケールの拡大縮小）。カーソル位置を中心にズームします。 [K2] |
| Ctrl + ホイール | キャンバス全体のズーム（PanCam によるカメラズーム）。チャート自身の価格スケールは変更しません。 [K4] |
| 右ドラッグ / 中ドラッグ | パン（キャンバス全体の移動）。 [K3] |
| 左ドラッグ | チャート内のパン。 [K3] |
| ダブルクリック | パン・ズームをリセットし、オートスケールを再有効化します。 [K3] |

- ホイールズームは時間軸方向（ローソク足の幅）と価格軸方向の両方を、カーソル直下の価格・時刻を固定したまま拡大縮小します。 [K2]
- ズーム倍率には上限・下限のクランプが設けられており、過剰なズームや潰れを防ぎます。 [K2]

## Ladder ペイン（Live モードのみ）

- 実行モードが Live（[モード](modes.md) を参照）のとき、チャートウィンドウの右側に板情報（Ladder）ペインが追加され、ウィンドウ全体が横方向に拡大します。 [K5]
- 構成は常に 21 行固定で、上半分が ask（売り）10 段、中央が `LAST`（最新約定価格）、下半分が bid（買い）10 段です。 [K5]
  - ask 行は赤系、bid 行は緑系の薄塗り背景、`LAST` 行は強調色で表示します。
  - 段数が不足する場合は `---` プレースホルダで埋めます。
- 各行は「価格 + 数量」の形式で表示します。 [K5]
- 板情報が無い場合（リプレイ・未購読など）は `No depth data` を表示します。 [K5]
- Ladder は銘柄ごとに独立しており、Live でない（リプレイ）モードに戻すとペインは閉じてコンパクトなチャート単独表示に戻ります。 [K5]
